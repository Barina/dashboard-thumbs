# Dashboard Thumbs

A community collection of **AI-generated and human-created thumbnails for dashboards**.

These thumbnails are designed for projects like Homarr, Dashy, Homepage and other dashboard tools.

Most thumbnails are animated WebP, but PNG and GIF formats are also supported.

## Features

- AI-generated and community created thumbnails
- Animated WebP support
- Multiple size variants (512, 256, 128)
- Automatic optimization via CI
- CDN delivery via jsDelivr
- Metadata support

## Usage

Thumbs can be loaded directly via CDN:

```
https://cdn.jsdelivr.net/gh/barina/dashboard-thumbs/webp/jellyfin.webp
```

Optimized version:

```
https://cdn.jsdelivr.net/gh/barina/dashboard-thumbs/webp/jellyfin@128.webp
```
